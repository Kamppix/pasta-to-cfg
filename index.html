Pasta to CFG

This page converts long texts into CFG files compatible with games such as Counter-Strike: Global Offensive. These CFG files create a command alias which, when called repeatedly (e.g. with a keybind set with bind <key> <alias>), outputs the entire text into the game's chat, in chunks. Note: if the command is called too fast, the server will most likely throttle the message rate, resulting in some chunks going missing.
